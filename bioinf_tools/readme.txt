To run the code, django 4.0.2 and biomart must be installed
pip install "Django==4.0.2"
pip install "biomart==0.9.2"
pip install "pandas==1.4.1"

To activate enviroment: source env/bin/activate (or the ..your own path/activate)

To run the server, navigate to the bioinf_tools directory and enter: python manage.py runserver
Then open http://127.0.0.1:8000/ in a browser

Upload vcf and bed files found in bioinf_tools/files
Press the upload button, and view results
